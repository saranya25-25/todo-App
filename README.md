📝 Todo App
A simple and interactive Todo List application built using HTML, CSS, and JavaScript.

This project helps users manage daily tasks with features like adding, editing, deleting, completing todos, and saving data using Local Storage.

🚀 Features

✅ Add new todos
✏️ Edit todos using double click
🗑️ Delete todos
☑️ Mark todos as completed
💾 Persistent storage using Local Storage
⌨️ Add todos using Enter key
🔄 Automatically loads saved todos after refresh
🛠️ Technologies Used

HTML5
CSS3
JavaScript (DOM Manipulation & Local Storage)
